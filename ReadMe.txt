INSTALLATION OF PROTOTYPE TO WORK IN LOCALHOST
**Combine all zipped files into one folder

1)If you don't have node.js, download node.js by searching it up on Google

2)use your node.js command line just search for it in your windows by typing the windows search bar "node.js"

3)On the project folder, using the node.js comman line type in npm install to install modules
required by Watson

4)Now you're able to run the prototype by typing "node app.js" on the command line. 

5)Now go to the specified localhost which is localhost:3700 and the prototype should be running.
